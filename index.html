How's This Book?

“How’s This book?” is a website that allows Oregon State University students (approximately 20,000 Users total) to publish Ratings for Books. It also allow Users to access details of Books, such as their Authors and Genres. The primary function of the website is to provide Users with a hub to store and access their Ratings for Books.

Outline of webpages:

Relative URL Description
/index.html Homepage with basic information about website
/books.html Browse books and add a book if it doesn't exist in database
/genres.html Browse books by genre and add a genre if it doesn't exist in database
/authors.html Browse books by author and add an author if he/she doesn't exist in database
/sample-book.html Browse all user ratings for a given book and add a rating
/users.html Browse and search for users
/sample-user.html Browser information and ratings for a given user
/account.html Log into user account and upon login, view/edit/delete existing ratings and view/edit account information
